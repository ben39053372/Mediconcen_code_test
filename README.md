## backend start
1. go to backend folder
  ```
  cd backend
  ```
1. copy .env.simple to .env. And fill in the information
2. run schema sql to init database schema
2. run
  ```
  yarn start / npm run start
  ```
1. go to front end
  ```
  cd ../frontend
  ```
2. config the setting.js change api baseUrl
3. run
  ```
  yarn start
  ```
